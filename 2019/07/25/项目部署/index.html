  2. 打包代码

    windows

    ​ zip rar ……

    linux

    ​ tar

    • -c 压缩
    • -x 解压
    • -v 显示过程
    • -f 指定文件名
    • -z gzip 是否压缩,不加只是单纯打包

  3. 传输代码

    • git

      • scp 本地文件路径 用户@远程主机ip:path

    • ftp

    • mount 共享挂载

    • rsync

      scp命令秘钥认证

      • 生成秘钥

      • 找到需要认证登录的服务器的ssh的配置文件

        默认在/etc/ssh/sshd_config中

        在配置文件 找到 AuthorizedKeysFile 文件位置

        %h 表示家目录

      • 将公钥的内容拷贝到对应的新建好的文件中

      • 可以重启远程ssh服务器 /etc/init.d/ssh restart

      • 可以重新验证
