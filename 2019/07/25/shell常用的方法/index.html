shell常用的方法

通过 定义 `ls` 这样的变量 可以对命令重复利用

  • shell 定义变量的时候不能有空格

  • shell 中 (){}[] 没有数据类型的含义

  • 全局变量 env 查看

    • 声明一个全局变量 export 变量名=变量值 临时声明
    • OLDPWD 上一个路径
    • 通过$变量名 获得变量值

  • 内置变量

    • 输入参数
      • $0 获取当前脚本名称
      • $# 获取当前脚本获取参数的个数
      • $n 获取当前脚本获取到的第n个参数
      • $? 获取上一次命令的执行情况,0表示成功 1表示失败 127 表示未找到
      • $$ 脚本运行时使用的进程号
      • $@ 获取所有的参数
    • 截取
      • ${var_name:start:n} 截取 start起始位置 n个数
      • $RANDOM 随机数 echo $RANDOM
    • 默认相关变量
      • var_name=$1 ${var_name:-defult} default 表示默认值,如果没有输入参数,default将会被使用
      • var_name=$1 ${var_name:+default} 将会无视输入的参数,直接输出设定好的默认值
    • 查看变量
      • $变量名 私下使用时
      • “$变量名” 调用变量时
      • “${变量名}” 脚本中,规范化作业时
    • unset date4 删除一个变量
    • 查找全局变量 env|grep date4
    • readonly 变量名 # 不能删除

  • shell 表达式

    • 验证操作

      • 方式一 [ 表达式 ] #注意空格
      • 方式二 test 表达式

    • 逻辑表达式

      • && #并 利用串联模式操作 有点像if语句
      • || #或 利用短路操作 if not 再执行

    • 文件表达式

      • -e 文件是否存在

      • -f 输入内容是否是个文件

      • -d 输入内容是否是个命令

      • -x 输入文件是否有执行权限

      • -r 判断文件是否刻度

      • -w 判断文件是否可写

      • 使用方式 [ ]

    • 数值表达式

      • -gt 大于
      • -lt 小于
      • -eq 等于
      • -ne 不等于

    • 字符串表达式

      • ==
      • !=
      • -z 判断字符串是否为0
      • -n 判断字符串是否不为0
