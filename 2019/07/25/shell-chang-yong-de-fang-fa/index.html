shell常用的方法

通过 定义 `ls` 这样的变量 可以对命令重复利用

  • shell 定义变量的时候不能有空格

  • shell 中 (){}[] 没有数据类型的含义

  • 全局变量 env 查看

    • 声明一个全局变量 export 变量名=变量值 临时声明
    • OLDPWD 上一个路径
    • 通过$变量名 获得变量值

  • 内置变量

    • 输入参数
      • $0 获取当前脚本名称
      • $# 获取当前脚本获取参数的个数
      • $n 获取当前脚本获取到的第n个参数
      • $? 获取上一次命令的执行情况,0表示成功 1表示失败 127 表示未找到
      • $$ 脚本运行时使用的进程号
      • $@ 获取所有的参数
    • 截取
      • ${var_name:start:n} 截取 start起始位置 n个数
      • $RANDOM 随机数 echo $RANDOM
    • 默认相关变量
      • var_name=$1 ${var_name:-defult} default 表示默认值,如果没有输入参数,default将会被使用
      • var_name=$1 ${var_name:+default} 将会无视输入的参数,直接输出设定好的默认值
    • 查看变量
      • $变量名 私下使用时
      • “$变量名” 调用变量时
      • “${变量名}” 脚本中,规范化作业时
    • unset date4 删除一个变量
    • 查找全局变量 env|grep date4
    • readonly 变量名 # 不能删除

  • shell 表达式

    • 验证操作

      • 方式一 [ 表达式 ] #注意空格
      • 方式二 test 表达式

    • 逻辑表达式

      • && #并 利用串联模式操作 有点像if语句
      • || #或 利用短路操作 if not 再执行

    • 文件表达式

      • -e 文件是否存在

      • -f 输入内容是否是个文件

      • -d 输入内容是否是个命令

      • -x 输入文件是否有执行权限

      • -r 判断文件是否刻度

      • -w 判断文件是否可写

      • 使用方式 [ ]

      • [ -f "${filename}" ] && echo "${filename} 是个文件"

    • 数值表达式

      • -gt 大于
      • -lt 小于
      • -eq 等于
      • -ne 不等于

    • 字符串表达式

      • ==
      • !=
      • -z 判断字符串是否为0
      • -n 判断字符串是否不为0

    • 流程控制语句

      • if语句

        • 单if 语句

          #!/bin/bash
arg=$#
if [ "${arg}" -eq 2 ]
then
    echo "参数个数为2"
fi

        • 双if 语句

          #!/bin/bash
arg=$#
if [ "${arg}" -eq 2 ]
then
    echo "参数个数为2"
else
    echo "参数个数不为2"
fi

        • 多if 语句

          #!/bin/bash
arg=$#
if [ "${arg}" -eq 2 ]
then
    echo "参数个数为2"
elif[ "${arg}" -eq 3] 
then
    echo "参数个数不为3"
else
    echo "end"
fi

      • case语句

        #!/bin/bash
cmd=$1
case "${cmd}" in
    start)
        echo "系统启动中"
        ;;
    stop)
        echo "系统停止中"
        ;;
    restart)
        echo "系统重启中"
        ;;
    *)
        echo "$0的使用方式是:$0 [ start|stop|restart ]"
        ;;
esac


  ```

  这里*表示其他是个通配符

- for循环

  ```bash
  #!/bin/bash

  arg_list=`ls`
  for i in "${arg_list}"
  do 
      echo "${i}"
  done
  ```

  echo 有自动换行功能

  shell中的运算操作

  - let n=n+1
  - n=\$((\$n+1))

- while循环

  ```bash
  #!/bin/bash
  n=0
  while [ "${n}" -lt 10 ] 
  do
      echo "${n}"
      let n=n+1
  done
  ```

- until循环

  ```bash
  #!/bin/bash
  n=0
  until [ "${n}" -lt 10 ] 
  do
      echo "${n}"
      let n=n+1
  done
  ```

- 复杂流程控制语句

  - 无参数

    ```bash
    #!/bin/bash

    func(){
        echo "这是个无参函数"
    }

    func
    ```

  - 有参数

    ```bash
    #!/bin/bash

    func(){
        args=$1
        echo "这是个有参函数:{args} "
    }

    func plus
    ```
